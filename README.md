#  Leukanos' Dotfiles
This is a collection of dotfiles and scripts I use for customizing OS X to my liking and setting up the software development tools I use on a day-to-day basis.

# Setup
1. Download and install [iterm2](https://www.iterm2.com/)
2. Download and install [Docker for Mac](https://hub.docker.com/editions/community/docker-ce-desktop-mac/)
3. Download this repository
4. Run `./install.sh`
